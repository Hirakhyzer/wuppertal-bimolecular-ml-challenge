# Conceptual research figures

These SVG assets are original documentation graphics for the repository README.

They illustrate:

- the conceptual end-to-end research architecture;
- the directed Coulomb-matrix and pair-feature construction;
- random-pair versus geometry-aware evaluation;
- the reproducible lifecycle from local inspection to evidence-based reporting.

They are **not dataset-derived plots**, do not visualize the supplied challenge data, and do not imply any achieved model performance. Data-driven figures belong only in `outputs/figures/` after local experiments have run.
